# NovaFlow Shop 🚀

Digital products for creators — Notion templates, Canva packs, automation workflows, and Lightroom presets.

🌐 **Live site:** [https://novaflowshops.vercel.app](https://novaflowshops.vercel.app)

## Tech Stack

- **Next.js** — React framework
- **Shopify Storefront API** — product & cart management
- **Tailwind CSS** — styling
- **Vercel** — hosting

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
